# Intelligent Patient Triage System 🏥

## 📌 Project Description
The Intelligent Patient Triage System is a Java console-based application 
that helps analyze patient symptoms and assess risk levels based on 
predefined medical logic.

This system helps in categorizing patients into:
- Low Risk
- Medium Risk
- High Risk

---

## 🚀 Features
- Patient data input
- Symptom-based risk calculation
- Console-based result display
- Structured using OOP concepts

---

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
